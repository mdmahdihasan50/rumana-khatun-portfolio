# Restore the complete project

1. Clone the repository.
2. Open `index.html` directly for local preview, or serve the directory with any static web server.
3. Upload the repository contents to `public_html` or another static hosting document root.

All required HTML, CSS, JavaScript and image assets are committed. No dependency installation or build step is required.

The ignored `index.zip` is only a duplicate archive of the same source and is not needed to restore the portfolio.
